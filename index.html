This page contains information about Patrick Chkeroff's Loom anonymous electronic trading system.

My blog has a Loom Category, which contains all the articles I've writen about Loom. The last time I updated this page, those were:

  • Loom Electronic Accounting System - an overview with links to tutorials and resources. I've since decided that "electronic trading system" is a better name.
  • Loom Security - my analysis of the security of the algorithms and database used by version 58 of the loom.cc source code.
  • Creating Your Own Loom Currency - an idea for keeping track of how much reserve of a real asset you have versus how much electronic currency you've issued backed by that asset.
  • Patrick's Loom Issuance Scheme - a better idea for tracking reserve assets.
  • Loom Three-folder Issuer Protocol - an even better idea for tracking reserve assets.

My PHP version of Patrick's Grid Tutorial is here. The tutorial is an example of using my LoomClient PHP class. I have also written a LoomClient Java class, with a simple command line testing interface. Its JavaDoc pages are here. Download this entire directory, including all the sources and Java binaries, at loomclient.tar.gz

You can browse my Git repository of the source at repo.or.cz/w/loomclient.git. To check out your own copy:

mkdir loomclient
cd loomclient
git init
git pull git://repo.or.cz/loomclient.git
