This page contains information about Patrick Chkeroff's Loom anonymous electronic trading system.

Applications:

  • grid-tutorial.php - my PHP version of Patrick's Grid Tutorial. The tutorial is an example of using my LoomClient PHP class. I have also written a LoomClient Java class, with a simple command line testing interface. Its JavaDoc pages are here.
  • ip.php - my iPhone Loom interface. Supports moving assets, and creating and renaming locations. I'll add location deletion, type maintenance, and skins for other mobile browsers, if there's demand.
